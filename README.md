# sunshine
 # 2D Platformer Prototype (Unity 6)  A small 2D platformer prototype built in **Unity 6**, demonstrating core gameplay mechanics including player movement, collectibles, and interactive world objects.  ## Features  * Player movement with running and jumping * Parallax scrolling background * Animated collectibles with scoring system * Key pickup system * Interactive chest that unlocks with a key * Animated door used as a level exit * Basic platformer level built using Unity Tilemap  ## Systems Implemented  * Unity **Input System** * **Animator Controller** for character and object animations * **Trigger-based interactions** for collectibles and objects * **Score Manager** for tracking collected items * **Audio feedback** for item collection * **Parallax background system** for layered environment movement  ## Built With  * Unity 6 * C# * Unity Tilemap system * Unity Animator  ## Purpose  This project was created as a **gameplay prototype** to practice implementing common platformer mechanics and interaction systems in Unity.  It demonstrates the foundation needed for building a complete 2D platformer game.
